Instalación del Sistema

Requisitos previos para la instalación:

1

Descargar Instalador

  • Acceda al portal de descargas autorizado del sistema
  • Ubique la sección de descargas para el sistema BDK
  • Descargue el archivo BDK_Instalador.exe a su computadora
Requisitos Mínimos: Windows Server 2016 o superior, SQL Server 2016+, 8 GB de RAM, 50 GB de espacio en disco
2

Ejecutar Instalador

  • Haga clic derecho sobre BDK_Instalador.exe y seleccione Ejecutar como administrador
  • Acepte los términos y condiciones de uso
  • Siga los pasos del asistente de instalación para seleccionar los componentes a instalar:
    • Servidor de aplicaciones
    • Consola administrativa
    • Clientes de estación de trabajo
